The SQL SUM() Function

The SUM() function sums up all the values in a given column or the values returned by an expression.

It’s a good introduction to SQL’s aggregate functions, so let’s dive right in!

Additional points about the SUM() function in SQL:

  • SUM() sums up all the values in a given column or the values returned by an expression.
  • You can add a WHERE clause to specify conditions.
  • Give the summarized column a name by using the AS keyword.
  • Use SUM() with GROUP BY to sum values for each group.
  • The parameter inside the SUM() function can also be an expression.

The SUM() function sums up all the values in a given column or the values returned by an expression.

Syntax

The basic syntax of the SQL SUM Function is as follows −

                SELECT SUM(column_name)
                FROM table_name
                WHERE condition;

FROM OrderDetails
