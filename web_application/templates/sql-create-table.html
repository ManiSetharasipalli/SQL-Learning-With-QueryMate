Create Table in SQL

In SQL, the CREATE TABLE statement is used to create a new table in a database.

It specifies the table name and defines the columns and their data types.

Additional points about creating tables in SQL:

  • The CREATE TABLE statement is used to create a new table in a database.
  • It specifies the table name, column names, and data types for each column.
  • Constraints such as PRIMARY KEY, FOREIGN KEY, NOT NULL, etc., can be added to the columns.
  • The table can also have additional properties like indexes, triggers, and permissions.

Syntax

            CREATE TABLE table_name (
                column1 datatype constraint,
                column2 datatype constraint,
                ...
            );

Benefits of CREATE TABLE

  • Structured data storage: CREATE TABLE enables the creation of structured tables for storing data.
  • Data integrity: It allows defining constraints to maintain data integrity, such as primary keys and foreign keys.
  • Flexibility: Tables can be customized with various data types, constraints, and additional properties to suit specific requirements.

Review of SQL Data Types

SQL data types define the type of data that can be stored in a column of a database table. Some common SQL data types include:

  • INT: Used for storing integer values.
  • VARCHAR(n): Used for storing variable-length character strings with a maximum length of 'n' characters.
  • DATE: Used for storing date values in the format 'YYYY-MM-DD'.
  • FLOAT: Used for storing floating-point numbers.
  • BOOLEAN: Used for storing boolean values ('TRUE' or 'FALSE').
