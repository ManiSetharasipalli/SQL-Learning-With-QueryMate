The SQL DISTINCT Keyword

The SQL DISTINCT keyword is used in conjunction with the SELECT statement to fetch unique records from a table.
We use DISTINCT keyword with the SELECT statetment when there is a need to avoid duplicate values present in any specific columns. When we use DISTINCT keyword, SELECT statement returns only the unique records available in the table. The SQL DISTINCT Keyword can be associated with SELECT statement to fetch unique records from single or multiple columns.

Syntax

SELECT DISTINCT column1, column2,.....columnN FROM table_name;

Where, column1, column2, etc. are the columns we want to retrieve the unique or distinct values from; and table_name represents the name of the table containing the data.

Explanation

  • DISTINCT: Keyword indicating the removal of duplicate rows.
  • column1, column2, .....columnN: Comma-separated list of columns to be retrieved.
  • table_name: Name of the table containing the data.

Examples

DISTINCT Keyword on Single Columns

We can use the DISTINCT keyword on a single column to retrieve all unique values in that column, i.e. with duplicates removed. This is often used to get a summary of the distinct values in a particular column or to eliminate redundant data.

Example Query

SQL Statement:

SELECT DISTINCT country
FROM Customers;

DISTINCT Keyword on Multiple Columns

We can also use the DISTINCT keyword on multiple columns to retrieve all unique combinations of values across those columns. This is often used to get a summary of distinct values in multiple columns, or to eliminate redundant data.

Example Query

SQL Statement:

SELECT DISTINCT country,city
FROM Customers;

DISTINCT Keyword with COUNT() Function

The COUNT() function is used to get the number of records returned by the SELECT query. The DISTINCT keyword ensures that only unique values are counted.

Example Query

SQL Statement:

SELECT COUNT(DISTINCT country) AS Number_Distinct_countries
FROM Customers;

Explanation:

  • COUNT(DISTINCT country): Counts the number of distinct values in the country column.
  • AS Number_Distinct_countries: Aliases the result as Number of Distinct countries for better readability in the output.
