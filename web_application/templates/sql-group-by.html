SQL GROUP BY Operation

The GROUP BY operation is used to group rows that have the same values into summary rows.

It is often used with aggregate functions (COUNT, MAX, MIN, SUM, AVG) to perform calculations on grouped data.

Additional points about the GROUP BY operation in SQL:

  • The GROUP BY operation groups rows that have the same values into summary rows.
  • It is used with aggregate functions to perform calculations on grouped data.
  • Each column in the SELECT list must either be part of the GROUP BY clause or an aggregate function.
  • GROUP BY can significantly affect query performance, especially on large datasets.

Syntax

            SELECT column_name(s), aggregate_function(column_name)
            FROM table_name
            GROUP BY column_name;

Examples

Example 1: Group Products by Category

Group products by category and count the number of products in each category:

SELECT category_id, COUNT(*) AS num_products
FROM products

Group customers by country and calculate the average age for each country:

Benefits of GROUP BY

  • Data summarization: GROUP BY allows for summarizing large datasets into meaningful aggregates.
  • Data analysis: It enables insightful analysis by grouping data based on specific criteria.
  • Performance optimization: Proper use of GROUP BY can optimize query performance by reducing the amount of data processed.
