SQL INSERT INTO

The SQL INSERT INTO statement is used to add new records (rows) to a table in a database.

It allows you to specify the table name and the values to be inserted into each column of the table.

Additional points about SQL INSERT INTO:

  • The INSERT INTO statement is used to add new rows to a table.
  • You can specify the table name and the columns for which you want to insert data.
  • The VALUES keyword is used to specify the values to be inserted into the columns.
  • You can insert data into one or multiple rows in a single INSERT INTO statement.

Syntax

The basic syntax of the SQL INSERT INTO statement is as follows:

            INSERT INTO table_name (column1, column2, column3, ...)
            VALUES (value1, value2, value3, ...);

Examples

Example 1: Inserting a Single Record

Insert a single record into the 'customers' table:

INSERT INTO customers (customer_name, age, country, city)
VALUES ('John Doe', 35, 'USA', 'New York');

Benefits of SQL INSERT INTO

  • Efficiency: INSERT INTO allows for efficient addition of new data records into database tables.
  • Flexibility: It provides flexibility in inserting data into specific columns or multiple rows in a single statement.
  • Accuracy: SQL INSERT INTO ensures accurate insertion of data into the designated columns, minimizing errors.
