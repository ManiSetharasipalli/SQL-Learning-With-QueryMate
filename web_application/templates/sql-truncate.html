SQL TRUNCATE Statement

The SQL TRUNCATE statement is used to remove all records from a table in a more efficient and faster way compared to the DELETE statement. It also resets any auto-increment fields to their initial values.

Syntax

            TRUNCATE TABLE table_name;

The TRUNCATE statement follows a simple syntax where you specify the name of the table from which you want to remove all records.

Example

Truncate all records from the 'students' table:

Benefits of TRUNCATE

  • Efficiency: TRUNCATE is faster and more efficient than using the DELETE statement to remove all records from a table, especially when dealing with large datasets.
  • Auto-increment Reset: TRUNCATE resets any auto-increment fields in the table to their initial values, whereas DELETE does not.
  • Transaction Log: TRUNCATE generates less transaction log compared to DELETE, which can help in conserving system resources.
