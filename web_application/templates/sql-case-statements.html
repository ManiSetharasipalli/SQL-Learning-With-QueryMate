CASE Statement in SQL

In SQL, the CASE statement is a powerful tool for conditional logic, allowing you to perform different actions based on specified conditions.

It provides a flexible way to manipulate data and generate calculated fields within a query.

Additional points about the CASE statement in SQL:

  • The CASE statement evaluates a list of conditions and returns one of several possible result expressions.
  • It can be used in SELECT, WHERE, and ORDER BY clauses to customize query results.
  • There are two forms of the CASE statement: simple CASE and searched CASE.

Syntax

            CASE
                WHEN condition1 THEN result1
                WHEN condition2 THEN result2
                ...
                ELSE default_result
            END

Example

Categorize Products

Categorize products as 'High', 'Medium', or 'Low' based on their prices:

                SELECT product_name, price,
                CASE
                    WHEN price >= 100 THEN 'High'
                    WHEN price >= 50 THEN 'Medium'
                    ELSE 'Low'
                END AS category
                FROM products;

Benefits of CASE Statement

  • Conditional Logic: The CASE statement allows for complex conditional logic within SQL queries.
  • Flexibility: It provides a flexible way to customize query results based on specified conditions.
  • Readability: Using the CASE statement can improve query readability by clearly expressing conditional transformations.
