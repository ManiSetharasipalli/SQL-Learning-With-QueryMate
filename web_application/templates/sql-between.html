SQL BETWEEN

The SQL BETWEEN operator is used to select values within a specified range.

It allows you to specify a range of values to filter the result set.

Additional points about SQL BETWEEN:

  • The BETWEEN operator selects values within a specified range.
  • It is inclusive, meaning it includes both the start and end values in the range.
  • BETWEEN is commonly used in WHERE clauses to filter data based on a range of values.
  • It is often used with numeric, date, or string values.

Syntax

The basic syntax of the SQL BETWEEN operator is as follows:

            SELECT column_name(s)
            FROM table_name
            WHERE column_name BETWEEN value1 AND value2;

Examples

Example 1: Filtering Orders by Order Date

Retrieve orders within a specific date range:

SELECT order_id, order_date
FROM orders
WHERE order_date BETWEEN '2022-01-01' AND '2022-01-31'; Query editor »

Advantages of SQL BETWEEN

  • Simplicity: BETWEEN simplifies query construction by allowing a concise range specification.
  • Precision: It provides precise filtering based on specified ranges, ensuring accurate data retrieval.
  • Flexibility: BETWEEN can be used with various data types, such as numeric, date, or string values, offering versatile filtering capabilities.
