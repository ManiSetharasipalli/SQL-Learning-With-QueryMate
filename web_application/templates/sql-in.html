SQL IN

The SQL IN operator allows you to specify multiple values in a WHERE clause.

It is used to reduce the use of multiple OR conditions in a query.

Additional points about SQL IN:

  • The IN operator allows you to specify multiple values in a WHERE clause.
  • It is often used with a subquery or a list of values enclosed in parentheses.
  • IN is equivalent to multiple OR conditions and can make queries more concise and readable.
  • It is useful for filtering rows based on a set of predefined values.

Syntax

The basic syntax of the SQL IN operator is as follows:

            SELECT column_name(s)
            FROM table_name
            WHERE column_name IN (value1, value2, ...);

Examples

Example 1: Using IN with a List of Values

Retrieve products with specific IDs:

SELECT product_name, price
FROM products
WHERE product_id IN (1, 3, 5); Query editor »

Example 2: Using IN with a Subquery

Retrieve orders from specific customers:

SELECT order_id, order_date
FROM orders
WHERE customer_id IN (SELECT customer_id FROM customers WHERE country = 'USA'); Query editor »

Benefits of SQL IN

  • Simplicity: IN reduces the complexity of queries by eliminating the need for multiple OR conditions.
  • Readability: It makes SQL queries more readable and understandable, especially when dealing with a large set of values.
  • Efficiency: IN can improve query performance by allowing the database to optimize the execution plan.
