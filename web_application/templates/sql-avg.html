The SQL AVG() Function

The AVG() function returns the average value of a numeric column.

It's commonly used to calculate the mean of a set of values in SQL.

Additional points about the AVG() function in SQL:

  • The AVG() function returns the average value of a numeric column.
  • You can add a WHERE clause to specify conditions.
  • Give the AVG column a name by using the AS keyword.
  • To list all records with a higher price than average, you can use the AVG() function in a subquery.
  • Use AVG() with GROUP BY to return the average value for each group.
  • Note: NULL values are ignored.

The AVG() function returns the average value of a numeric column.

Syntax

The basic syntax of the SQL AVG Function is as follows −

                SELECT AVG(column_name)
                FROM table_name
                WHERE condition;
