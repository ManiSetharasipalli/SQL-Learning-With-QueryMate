SQL INNER JOIN Operation

The INNER JOIN operation is used to combine rows from two or more tables based on a related column between them.

It retrieves rows when there is at least one match in both tables.

Additional points about the INNER JOIN operation in SQL:

  • The INNER JOIN operation is used to retrieve rows from multiple tables based on a related column.
  • It combines rows from two tables where there is a match between the specified columns.
  • You can add conditions using the WHERE clause to further filter the result set.
  • Aliases can be used for table names to make the SQL code more readable.
  • INNER JOIN is commonly used in scenarios where data needs to be combined from multiple tables.
  • Aliases are used to provide temporary names for database tables or columns.
  • The ON keyword specifies the condition for joining the tables.

The INNER JOIN operation is used to retrieve rows from multiple tables based on a related column.

Syntax

The basic syntax of the SQL INNER JOIN Operation is as follows:

                SELECT column_name(s)
                FROM table1
                INNER JOIN table2
                ON table1.column_name = table2.column_name;

Examples

Example 1: Retrieving Product Details with Categories

Retrieve product details along with their categories:

SELECT p.product_id, p.product_name, p.price, c.category_name
FROM products p
INNER JOIN categories c
ON p.category_id = c.category_id; Query editor »

In this example, we retrieve product details from the 'products' table and category names from the 'categories' table. The INNER JOIN operation is used to combine rows from both tables where the 'category_id' in the 'products' table matches the 'category_id' in the 'categories' table.

Example 2: Retrieving Order Details with Products

Retrieve order details along with the products ordered:

SELECT o.order_id, p.product_name, od.quantity, od.unit_price
FROM orderdetails od
INNER JOIN orders o
ON od.order_id = o.order_id
INNER JOIN products p
ON od.product_id = p.product_id; Query editor »

This example retrieves order details along with the products ordered by joining the `orderdetails`, `orders`, and `products` tables based on the `order_id` and `product_id` columns.

Example 3: Retrieving Customer Details with Orders

Retrieve customer details along with their orders:

SELECT c.customer_id, c.customer_name, o.order_id, o.order_date
FROM customers c
INNER JOIN orders o
ON c.customer_id = o.customer_id; Query editor »

This example retrieves customer details along with their orders by joining the `customers` and `orders` tables based on the `customer_id` column.
