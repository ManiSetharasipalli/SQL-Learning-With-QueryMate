The SQL SELECT Statement

The SQL SELECT Statement is used to fetch the data from a database table which returns this data in the form of a table. These tables are called result-sets.

Syntax

The basic syntax of the SELECT Query is as follows −

            SELECT column1, column2, columnN FROM table_name;

Example Query

SQL Statement:

SELECT * FROM Customers; Query editor »

Try this example in Query editor.

Examples

Query 1: Select All
SELECT * FROM customers; Query editor »

Try this example in Query editor.

It retrieves all the data from every column in the table and presents it in a result set.

Here's a breakdown:

  • SELECT * :- keyword instructs the database to retrieve all columns from the specified table.

  • FROM customers :- This clause specifies the name of the table from which data needs to be retrieved.
  • ; :- Semicolon to terminate the statement.
Query 2: Select Specific Columns
SELECT Customer_name,age FROM customers; Query editor »

Try this example in Query editor.

This query selects specific columns (name, age) from the table named customers
It retrieves only the data for those two columns and presents them in a result set.

Here's a breakdown:

  • SELECT Customer_name, age :- This clause specifies the individual columns that we want to retrieve data from. You can list multiple columns separated by commas.
  • FROM customers :- Similar to the first query, this clause specifies the target table
  • ; :- Semicolon to terminate the statement.
