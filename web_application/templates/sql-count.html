The SQL COUNT() Function

The COUNT() function returns the number of rows that match a specified criterion.

Aggregate functions are often used with the GROUP BY clause of the SELECT statement. The GROUP BY clause splits the result-set into groups of values and the aggregate function can be used to return a single value for each group.

Additional points about the COUNT() function in SQL:

  • COUNT() returns the number of rows in a set
  • You can specify a column name instead of the asterisk symbol (*).
  • If you specify a column name instead of (*), NULL values will not be counted.
  • You can add a WHERE clause to specify conditions.
  • You can ignore duplicates by using the DISTINCT keyword in the COUNT() function.
  • You can give the counted column a name by using the AS keyword.
  • Use COUNT() with GROUP BY to count records for each group.

The COUNT() returns the number of rows that match a specified criterion.

Syntax

The basic syntax of the SQL COUNT Function is as follows −

                SELECT COUNT(column_name)

                FROM table_name

                WHERE condition;
